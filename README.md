## 📦 Instalação

Clone o projeto e instale as dependências:

```bash
git clone https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git
cd comando-dos-guri
npm install
```

# 🚀 Funcionalidades

- CRUD de usuários

- Integração com banco via Prisma (Schema: ./prisma/schema)

- Autenticação JWT

- Estrutura pronta para testes com Jest
